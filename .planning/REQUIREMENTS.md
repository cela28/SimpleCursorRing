# Requirements: SimpleCursorRing

**Defined:** 2026-02-03
**Core Value:** A visible, customizable ring follows the cursor — helping players track their mouse position during gameplay, especially in combat.

## v1 Requirements

Requirements for initial release.

### Core Ring

- [ ] **RING-01**: Ring displays around cursor and follows mouse movement
- [ ] **RING-02**: Ring size adjustable via slider (20-200px)
- [ ] **RING-03**: Ring color customizable via color picker
- [ ] **RING-04**: Option to use player's class color for ring

### Settings

- [ ] **SETT-01**: Settings accessible via /simplecursorring slash command
- [ ] **SETT-02**: Settings accessible via Interface Options addon panel

## v2 Requirements

Deferred to future release. Tracked but not in current roadmap.

### Visibility Modes

- **VIS-01**: Option to show ring only in combat
- **VIS-02**: Option to show ring only while right-clicking

### Combat Features

- **COMB-01**: Combat override — separate size/color when in combat
- **COMB-02**: Combat overlay — optional second ring during combat with own size/color

## Out of Scope

Explicitly excluded. Documented to prevent scope creep.

| Feature | Reason |
|---------|--------|
| Center dot | User explicitly excluded — not wanted in any version |
| Mouse trail | User requested ring only |
| Other EnhanceQoL features | Single-purpose addon |
| Multi-language localization | English only for v1 |
